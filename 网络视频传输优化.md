## 比较

两种解决方案各有优劣，适用于不同场景。相比于硬件设备解决方案，算法解决方案对内容发布者更为适合，毕竟对内容订阅者没有太多的限制。

| 思路         | 优势   | 劣势                         |
| ------------ | ------ | ---------------------------- |
| 算法         | 低时延 | 受限于内容发布者终端设备算力 |
| 硬件设备支持 |        | 时延较算法方案较高           |
